## 🚀 Featured Projects

### 🏥 [AI-Infused Gamified Learning Platform — Healthcare](https://github.com/Bhoomika00/AI-infused-gamified-learning-website-healthcare)
> MSc Dissertation @ Oxford Brookes University

AI-powered e-learning platform for healthcare frontline workers. Automatically generates personalised courses, lessons, and quiz assessments (MCQ, True/False, Scenario-based) from raw training content using LLMs and NLP pipelines. Built as a full-stack application.

`Python` `OpenAI API` `LangChain` `NLP` `FastAPI` `MySQL` `JavaScript`

---

### 🧠 [Parkinson's Disease Detection](https://github.com/Bhoomika00/Parkisons-disease-detection)
> ML + Computer Vision

AI-based early screening system using spiral and wave handwriting images. Compared 6 ML algorithms — achieved **86% accuracy** with Random Forest. Full image preprocessing pipeline with OpenCV, web interface, and MongoDB integration for result storage.

`Python` `Scikit-learn` `XGBoost` `OpenCV` `MongoDB` `HTML/CSS`

---

### 📧 [Email Spam Classifier](https://github.com/Bhoomika00/Email-spam)
> NLP Text Classification Pipeline

NLP classification pipeline using TF-IDF vectorisation and Scikit-learn to detect spam emails. Trained, validated, and serialised model deployed as a Python web application.

`Python` `Scikit-learn` `NLTK` `TF-IDF` `Flask`

---
### 🛒 [Market Basket Analysis & Customer Segmentation](https://github.com/Bhoomika00/market-basket-analysis-customer-segmentation)
> Unsupervised ML | MSc Coursework @ Oxford Brookes

Applied association rule mining (Apriori) and customer segmentation (K-Means) to real retail transaction data to uncover purchasing patterns and customer groups. Used PCA for dimensionality reduction and cluster visualisation.

`Python` `Scikit-learn` `mlxtend` `Apriori` `K-Means` `PCA` `Pandas` `Matplotlib`
